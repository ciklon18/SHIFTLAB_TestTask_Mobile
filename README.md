# Тестовое задание для лаборатории ШИФТ

Простое приложение со страницей регистрации и страницей приветствия


## Используемые технологии
- Jetpack Compose
- MVVM
- Clean Architecture
- Koin
- Kotlin coroutines
- Shared preferences
